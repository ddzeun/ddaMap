# 🚴‍♂️ 따맵 (DDA-MAP)

> 서울시 공공자전거 '따릉이' 이용자를 위한 경로 탐색 서비스


## 🌟 1. 프로젝트 개요

### 1-1. 서비스 소개

"출발지와 도착지만 입력하면, 가장 가까운 따릉이 대여소를 포함한 최적 경로를 찾아주는 서비스" 입니다.

지도 앱과 따릉이 앱을 번갈아 사용하는 불편함을 해결하고, 실제 이동시간과 경로를 예측하여 사용자 편의성을 높이는 것을 목표로 합니다.

### 1-2. 핵심 기능

- 🗺️ **최적 경로 추천**: 가장 가까운 대여소/반납소를 경유하는 '도보+자전거+도보' 통합 경로 제공
- ⏰ **예상 소요 시간**: 전체 경로에 대한 예상 소요 시간을 분 단위로 제공
- 🚲 **실시간 정보**: 서울시 공공데이터 API와 연동하여 대여소별 실시간 자전거 현황 제공
- 🔍 **대여소 검색**: 이름 및 주소 기반으로 원하는 대여소 정보 검색

## 🎯 2. 기획 배경 및 목표

### 📌 주제 선정 배경

따릉이로 출발지부터 목적지까지의 예상 소요시간을 알고 싶을 경우
목적지와 따릉이 반납소는 대부분 다르기 때문에 감으로 짐작해야 했고,
정확히 알고 싶을 때는 지도 앱에서 경유지를 여러 번 추가하는 번거로운 과정을 거쳐야만 했다.

이러한 반복적인 불편함을 줄여보고자 이 프로젝트를 시작하게 되었다.

---

### 🎯 목표

- **서비스가 해결하고자 하는 문제**

  → 따릉이 앱에는 경로 탐색 기능이 없어, 실제 이동 시간을 예측하거나 경로를 보기 어려움

- **이 서비스를 사용하는 이유**

  → 출발지와 도착지만 입력하면, 가장 가까운 대여소를 자동으로 포함해
  **실제 이동 경로와 예상 시간까지 제공**받을 수 있기 때문

- **서비스 핵심 가치**

  → **출발지·도착지만 입력하면 '따릉이 대여소 포함 경로'를 자동 추천받는 기능**

- **타깃 사용자**

  → 따릉이와 지도 앱을 번갈아 쓰는 사용자

  → 대중교통보다 따릉이를 선호하지만, 이동 시간과 경로 확인이 번거로운 사용자

## 📑 3. 주요 기능 및 요구사항

| 기능 ID | 기능명 | 설명 |
| --- | --- | --- |
| F-01 | 대여소 목록 보기 | 현재 위치 기준 지도 화면을 띄우고, 근처 대여소의 잔여 자전거 수를 보여준다. ‘목록 보기’ 버튼을 누르면 리스트 형태로도 확인할 수 있다. |
| F-02 | 지도 탐색 | 지도를 드래그/확대하여 현재 위치 외의 지역을 탐색할 수 있다. 탐색 영역에 따라 대여소 정보가 갱신된다. |
| F-03 | 대여소 정보 새로고침 | 지도 화면에서 ‘새로고침’ 버튼을 누르면 현재 보이는 지역의 대여소 잔여 자전거 수 정보를 최신화한다. |
| F-04 | 현재 위치로 이동 | 지도 탐색 중 ‘현재 위치로 이동’ 버튼을 눌러 다시 내 위치 중심으로 지도를 리셋할 수 있다. |
| F-05 | 대여소 상세 정보 | 지도에 표시된 마커를 클릭하면 해당 대여소의 자전거 현황, 주소를 확인할 수 있고, 즐겨찾기 추가 및 길찾기도 가능하다. |
| F-06 | 검색 | 대여소/특정 장소, 주소 등을 검색할 수 있다. |
| F-07 | 경로 탐색 | 출발 위치(기본값: 현재 위치)와 도착지를 입력하면, 선택한 경로 타입(따찾기 / 도보 / 자전거)에 따라 최적 경로를 추천하고 예상 소요시간을 보여준다. 따찾기 선택 시, 자동으로 대여/반납 대여소를 포함한 경로가 생성된다. |

## 🔗 5. API 명세
> [👉 전체 API 명세](https://github.com/ddzeun/ddaMap/blob/feature/1-docs/docs/API.md)


| 기능 ID | 기능명 | API | 비고 |
| --- | --- | --- | --- |
| **F-01** | 대여소 목록 보기 | `GET /api/v1/stations` | 지도 중심 좌표 기준으로 대여소 목록 반환 |
| **F-02** | 지도 탐색 | `GET /api/v1/stations` | 중심 좌표 변경 시 같은 API로 요청 |
| **F-03** | 대여소 정보 새로고침 | `GET /api/v1/stations` | 같은 API, 단 갱신시간 표시만 다름 |
| **F-04** | 현재 위치로 이동 | ❌ API 없음 | 클라이언트에서 위치 재요청만 수행 |
| **F-05** | 대여소 상세 정보 | `GET /api/v1/stations/{id}` | 마커 클릭 시 해당 대여소 상세 조회 |
| **F-06** | 대여소 검색 | `GET /api/v1/stations?keyword={keyword}` | 키워드 기반 대여소 검색 |
| **F-07** | 경로 탐색 | `POST /api/v1/routes` | 출발지, 도착지, 경로 타입 등을 포함한 경로 요청 |
