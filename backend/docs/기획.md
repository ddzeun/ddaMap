# 🚲 따맵: 따릉이 길찾기 서비스 기획서

---

## 1. 📝 프로젝트 개요

* **프로젝트명**: 따맵 (DDA-Map)
* **개발자**: [https://github.com/ddzeun](https://github.com/ddzeun)
* **목표**: 출발지와 목적지만 입력하면, 적절한 따릉이 대여소를 자동으로 경유지에 포함해 길찾기 경로를 안내하는 웹 서비스

### 📌 기획 배경

* 서울시 따릉이 앱은 **대여소 위치만 제공**하고, **길찾기 기능은 없음**
* 일반 지도 앱은 따릉이 대여소를 **직접 경유지로 설정해야 해 불편함**
* 사용자가 **출발지와 도착지만 입력**하면, **가장 적절한 대여소와 반납소를 찾아 자동으로 경유지 포함**한 길찾기를 제공하면 훨씬 더 **편리한 사용자 경험 제공 가능**

---

## 2. 🎯 핵심 기능

| 기능 이름            | 설명                                                    |
|------------------|-------------------------------------------------------|
| 출발지·도착지 입력       | 사용자가 주소를 입력하거나 지도를 클릭하여 출발/도착지 설정                     |
| 대여소 자동 탐색        | 출발지와 도착지에서 가까운 대여소와 반납소를 자동으로 탐색                      |
| 경로 생성 (자동 경유 포함) | 자동으로 선택된 대여소를 포함해 카카오맵 API로 최적 경로 생성                  |
| 대여소 정보 제공        | 대여소 이름, 주소, 실시간 자전거 수, 거리 정보 제공                       |
| (후순위) 조건별 대여소 선택 | 거리 외에도 경사, 대여소 방향, 자전거 수, 경로상 위치 등 다양한 기준으로 필터링 지원 예정 |
| (선택) 소셜 로그인 기능   | 외부 계정을 통한 간편 로그인                             |
| (선택) 즐겨찾기 등록 기능  | 자주 이용하는 경로/대여소를 저장 및 불러오기 가능                          |

## 3. 🛠 기술 스택 및 도구

| ---------- | -------------------------------- |
| Language   | Java 17                          |

## 4. 📐 시스템 구성도

```plaintext
[출발지, 목적지 사용자 입력]
  ↓
[주소 → 좌표 변환]
  ↓
[정류소 탐색 - 따릉이 API]
  ↓
[가까운 정류소 자동 경유로 경로 생성 - 카카오맵 API]
  ↓
[최종 경로 및 정류소 정보 제공]
```

---

## 5. 🔄 API 설계 초안

| HTTP Method | Endpoint                   | 설명                            |
| ----------- | -------------------------- |-------------------------------|
| `GET`       | `/stations/near?lat=&lng=` | 좌표 기반 주변 대여소 목록 조회            |
| `POST`      | `/routes`                  | 출발지/도착지 기반 경로 생성 (대여소/반납소 포함) |
| `GET`       | `/stations/{id}`           | 특정 대여소 상세 정보 조회               |
| `POST` (선택) | `/favorites`               | 즐겨찾기 등록                       |
| `GET`  (선택) | `/favorites`               | 즐겨찾기 목록 조회                    |
